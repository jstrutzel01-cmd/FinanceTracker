# Finance Tracker App

## ðŸ“± Project Overview
Finance Tracker is an Android application built using **Jetpack Compose** and **Material Design 3**.  
The app allows users to track their income and expenses, organize transactions by category, and view their financial activity in a clean and user-friendly interface.

The project demonstrates modern Android development practices including Compose UI, Navigation Component, Room database, Firebase Authentication, and MVVM architecture.
